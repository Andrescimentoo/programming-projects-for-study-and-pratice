Nesse módulo do meu curso do DevClub aprendendo sobre JS, aprendi a como manipular o DOM. O DOM é a árvore de elementos de uma 
página HTML. Com o DOM podemos dar um caminho para o JavaScript operar e manipular um documento HTML.

O DOM vem com métodos que nos permite ultilizar na montagem do código em JavaScript, a fim de nos possibilitar a captura e a manipulação 
dos elemntos HTML de umma pagina web.

Ao longo dessa Pasta sobre DOM você verá alguns métrodos do DOM específicos estudados, e alguns pequenos desafios para reforçar
ainda mais o aprendizado.
